WebGPU golden parity harness

Compares WebGL and WebGPU for fixed MVP graphs. The test is running that comparison: use Run parity below (or the Console). This page only shows instructions until you run it.

  1. Serve via Vite: npm run dev or npm run preview (app base is /ShaderNoice/).
  2. Open this page under that base, e.g. http://localhost:3000/ShaderNoice/webgpu-golden-harness.html (port may differ).
  3. Click Run parity (all fixtures) above, or in the Console: await window.__webgpuGolden.runAll(). If the API is missing, check Console/Network for a failed script (wrong path or file://).

Notes: Without await you only see a pending Promise. window.__webgpuGolden?.runAll() returns undefined if the harness did not load; use __WEBGPU_GOLDEN_READY to verify. Headless/scripted runs: npm run test:webgpu-golden.
